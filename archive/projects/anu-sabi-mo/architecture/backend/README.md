# Backend Architecture

## Purpose

This directory documents the backend architecture for Anu Sabi.

The backend is responsible for all business logic, authentication, multiplayer state, game rules, persistence, and integrations.

The backend should remain independent of the frontend so that future clients (mobile apps, desktop apps, admin portals, APIs) can reuse the same services.

---

## Scope

- Business Logic
- Authentication
- Authorization
- REST APIs
- WebSocket Events
- Game Engine
- Matchmaking
- Economy
- Notifications
- Moderation
- Analytics
