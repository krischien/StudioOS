# 🗺️ Anu Sabi Roadmap

> *Building Anu Sabi one milestone at a time.*

This roadmap outlines the long-term direction of the Anu Sabi project. It serves as a planning guide rather than a strict schedule. Features and priorities may evolve as development progresses.

---

# Roadmap Philosophy

The roadmap follows three guiding principles:

- Build a polished core experience before expanding features.
- Prioritize player enjoyment over feature quantity.
- Ensure every new system supports long-term replayability.

---

# Development Phases

## ✅ Phase 1 — Foundation

**Status:** In Progress

### Goals

- Establish project architecture
- Build the core gameplay loop
- Design responsive mobile UI
- Create the documentation knowledge base
- Implement local persistence
- Build reusable game components

### Deliverables

- Core gameplay
- Coin system
- Badge system
- Hint system
- Daily streak tracking
- Content packs
- Professional documentation

---

## 🟡 Phase 2 — Gameplay Expansion

**Status:** Planned

### Goals

Expand the gameplay experience without increasing unnecessary complexity.

### Planned Features

- More phrase categories
- Improved balancing
- Better progression
- Achievement expansion
- Additional badges
- Improved animations
- Accessibility improvements
- Enhanced sound effects
- Better onboarding

---

## 🔵 Phase 3 — Content Platform

**Status:** Planned

### Goals

Transform Anu Sabi into a scalable content platform.

### Planned Features

- Seasonal content
- Holiday events
- Limited-time phrase packs
- Community-inspired content
- Dynamic content updates
- Content management tools

---

## 🔵 Phase 4 — AI-Assisted Development

**Status:** Vision

### Goals

Leverage AI to improve content quality and development efficiency.

### Potential Features

- AI-assisted phrase generation
- AI difficulty suggestions
- Duplicate detection
- Category recommendations
- Automated quality checks
- Internal content review assistant

Human review will remain mandatory for all published content.

---

## 🔵 Phase 5 — Player Experience

**Status:** Vision

Potential improvements include:

- Player profiles
- Cloud synchronization
- Statistics dashboard
- Expanded achievements
- Enhanced customization
- Accessibility enhancements

---

## 🔵 Phase 6 — Premium Experience

**Status:** Vision

Potential features:

- Premium phrase packs
- Cosmetic themes
- Exclusive badges
- Optional subscriptions
- Expanded customization

Monetization should never compromise fair gameplay.

## Long-Term Vision

Anu Sabi aims to become:

- The definitive Filipino party word game.
- A scalable platform for culturally relevant content.
- A showcase project demonstrating professional game architecture and documentation.
- The flagship project of StudioOS.

---

# Guiding Principles

Throughout development we will prioritize:

- Simplicity
- Replayability
- Accessibility
- Maintainability
- Quality over quantity
- Documentation-first development
