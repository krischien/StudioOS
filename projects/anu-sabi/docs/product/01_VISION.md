> **Last updated:** 2026-07-08

## What is Anu-Sabi?

**Anu-Sabi** is the project name. Players see **SabiMo?** on the home screen and in the UI. The mobile app is packaged as **Sabi Mo Ano** (`com.sabimoano.game`).

It is a **word-guessing game** with one distinctive mechanic:

> You see phonetic gibberish. You read it aloud (or in your head). You type the real phrase before time runs out.

| In-app voice | Copy |
|--------------|------|
| Home | *"Can you decode today's gibberish?"* |
| Game | *"Read it aloud to find the real phrase..."* |
| Correct answer | *"Tama!"* (Tagalog for "correct") |

Content spans **Filipino (Pinoy)** phrases and **world** phrases — songs, quotes, and everyday lines rendered as playful sound-alike text.

**What the product design implies today:**

1. **Celebrate language play** — Especially Filipino expressions, wordplay, and how things *sound* when spoken.
2. **Offer quick, satisfying puzzles** — Sessions fit a commute, a break, or a waiting room.
3. **Grow without forcing accounts** — Phase 1 is offline and local; no login wall before the first game.

## What makes it different from other word games?

| Typical word game | Anu-Sabi / SabiMo? |
|-------------------|---------------------|
| See letters / crossword clues | See **phonetic gibberish** meant to be **spoken** |
| Often English-only | **Pinoy-first content** with a parallel **world** pool |
| Dictionary or spelling focus | **Phrase recognition** — whole lines, lyrics, sayings |
| Often online-required | **Offline-first** (Phase 1) — play without network |
| Grind for levels | **Skill streaks** raise score multiplier; badges reward mastery moments |

## What experience should players have?

### Moment-to-moment

- **Curiosity** — *"What is this gibberish supposed to be?"*
- **Delight** — The "aha" when the real phrase lands, especially after saying it aloud
- **Tension** — Timer pressure without punishing exploration (wrong answers allow retry)
- **Pride** — Streak multipliers, badge unlocks, star ratings on the end screen

### Session-level

- **Short and complete** — 10-round mode delivers a full arc in a few minutes
- **Fair** — Hints cost resources; skips reveal but reset streaks
- **Rewarding** — Coins and badges acknowledge effort even on imperfect runs

### Long-term

- **Habit** — Daily Streak login rewards and Daily Challenge progress encourage return
- **Mastery** — Rank rises with badges; endless mode and hard difficulty reward deep players
- **Identity** — Display name, medal collection, and game history build a light personal profile

## What emotions should the game create?

| Emotion | When |
|---------|------|
| **Playfulness** | Reading silly gibberish aloud |
| **Surprise** | Unexpected connection between sounds and real phrases |
| **Accomplishment** | Correct answers, streaks, badge popups |
| **Anticipation** | Timer countdown, mystery chest on Day 5 of login streak |
| **Belonging** (future) | Friends, leagues, party mode — **Planned**, not live |

The visual tone supports this: bright gradients, bouncy buttons, confetti on strong runs, flame motifs for streaks.

## What does success look like?

### Phase 1 success (current milestone — offline playable)

- A new player can **install, play a full session, and earn rewards without internet**
- Core loop is **fun within the first 60 seconds** (first phrase → first correct answer)
- Players **return the next day** (Daily Streak / Daily Challenge engagement)
- App is **shippable** on mobile stores — TBD: release criteria require confirmation
