## Quick summary

**SabiMo?** is a mobile word game where players read phonetic gibberish aloud to decode real Filipino and world phrases.

**Current focus:** Phase 1 — **offline playable** (single-player, no network required).

**Phases:** 1 Offline → 2 Online → 3 Party (Bluetooth).
