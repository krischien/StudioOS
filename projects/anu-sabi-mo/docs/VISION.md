# Purpose

This document defines the long-term vision of **Anu Sabi**. It serves as the foundation for all design, development, content, business, and publishing decisions.

Whenever a new feature is proposed, this document should answer one question:

> "Does this move Anu Sabi closer to its vision?"

If the answer is no, the feature should be reconsidered.

---

# Vision Statement

Anu Sabi aims to become the most entertaining Filipino social party game where players create hilarious conversations, unexpected situations, and unforgettable memories through creativity, humor, and imagination.

Rather than testing knowledge or speed, Anu Sabi celebrates personality, storytelling, and the wonderfully unpredictable ways people communicate.

Every match should create moments that players want to talk about long after the game ends.

---

# Mission

To build a social game that:

- Makes people laugh.
- Encourages creativity.
- Brings friends and families together.
- Creates highly replayable experiences.
- Can be enjoyed by players of all ages.
- Continually evolves through new content and game modes.

---

# Core Philosophy

Anu Sabi is not about finding the correct answer.

It is about creating the most entertaining answer.

Players should never feel punished for being weird, imaginative, dramatic, or completely unexpected.

The game rewards creativity over correctness.

# Design Principles

Every feature added to Anu Sabi should support at least one of these principles.

## Accessible

Easy to learn within minutes.

## Social First

Best experienced with multiple players.

## Replayable

No two games should feel identical.

## Funny

Humor is the primary emotional goal.

## Creative

Players create content instead of simply consuming it.

## Expandable

The game should easily support future themes, categories, and game modes.

---

# Unique Selling Points

Anu Sabi differentiates itself through several key features.

## Rambler Engine

A dynamic system that generates unpredictable prompts, situations, twists, and combinations to maximize replayability.

## Filipino First

Designed with Filipino humor, language, culture, and everyday experiences at its core while remaining adaptable for international audiences.

## Endless Content

New content packs can be released without changing the core game.

## Community Potential

Future versions may allow players to submit, rate, and share prompts, creating a growing library of community-driven content.

# Success Metrics

Success is measured not only by downloads but also by player engagement.

Key indicators include:

- Daily Active Users (DAU)
- Monthly Active Users (MAU)
- Session Length
- Player Retention
- Match Completion Rate
- Content Pack Adoption
- Community Growth
- Positive Reviews
- Organic Sharing
- User-Generated Content

# What Anu Sabi Is

- A social party game.
- A creativity game.
- A conversation starter.
- A replayable entertainment platform.
- A foundation for future content.

---

# What Anu Sabi Is Not

- A trivia game.
- A quiz app.
- A knowledge competition.
- A fast reflex game.
- A traditional board game adaptation.

---

# Future Vision

The long-term ambition is for Anu Sabi to become more than a single game.

It should evolve into a platform capable of supporting:

- Expansion packs.
- Seasonal events.
- Community-created content.
- AI-assisted prompt generation.
- Tournament modes.
- Livestream integration.
- Educational adaptations.
- Corporate team-building editions.

---

# Guiding Question

Every future feature should answer:

> Does this make the game more fun, more social, or more replayable?

If not, reconsider adding it.
