# 1. Purpose

This document defines the modular content system used by Anu Sabi.

Content Packs allow the game to expand continuously through new prompts, themes, mechanics, seasonal events, and community creations without changing the core gameplay.

The system is designed for long-term scalability.

---

# 2. Vision

Content Packs transform Anu Sabi from a single party game into a living platform.

Players should always feel there is something new to discover.

---

# 3. Design Goals

Content Packs should:

* Expand replayability.
* Encourage experimentation.
* Support different player groups.
* Celebrate Filipino culture.
* Allow future international localization.
* Be easy to create and maintain.
* Support seasonal releases.

---

# 4. Content Architecture

Every pack follows a common structure.

Example:

```text
Content Pack
    │
    ├── Categories
    ├── Prompts
    ├── Modifiers
    ├── Roles
    ├── Twists
    ├── Special Rules
    ├── Rewards
    └── Cosmetics
```

This consistency allows the Rambler Engine to mix content intelligently.

# 14. Rambler Engine Integration

Content Packs are not isolated.

The Rambler Engine can combine elements from multiple packs.

Example:

```text
Theme:
Family Chaos

+

Modifier:
Halloween

+

Role:
Barangay Captain

+

Twist:
Speak Only in Song Lyrics
```

The result is a completely new experience without requiring a unique prompt for every combination.

This modularity is a core strength of Anu Sabi.

# 20. Decision Rationale

### Why modular content?

Modular design allows exponential growth by combining existing content in new ways, reducing development effort while increasing replayability.

### Why cultural packs?

Filipino culture is one of Anu Sabi's strongest differentiators. Dedicated packs celebrate that identity while allowing room for global expansion.

### Why challenge packs?

Changing the rules creates fresh experiences without requiring thousands of additional prompts.

---

# 21. Success Metrics

The Content Pack system succeeds when:

* Players regularly try new packs.
* Replayability remains high.
* New releases increase player retention.
* Community demand for new themes grows.
* Packs remain easy to maintain and expand.
